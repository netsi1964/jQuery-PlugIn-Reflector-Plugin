jQuery-PlugIn-Reflector-Plugin by netsi1964

add "plugInReflector()" to a jQuery element to reflect on internal jQuery objects.

This is a jQuery plug-in which can do some reflection on jQuerified elements. This can be very usefull when you try to explore possibilities of say image galleries. plugInReflector will show you in a structured way the settings of such galleries. I will keep developing this plug-in.

Install

add "plugInReflector()" to a jQuery element to reflect on internal jQuery objects.
